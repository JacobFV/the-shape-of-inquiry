# Typography system

Vertical short format, 1080x1920.

- Serif for thesis statements and major claims.
- Mono for instrumentation, equations, event types, metrics.
- Small uppercase technical labels for physical affordances.
- No large paragraph blocks held for long durations.

Color:
- blackbody base: #050505
- tungsten: #d9ae72
- sensor blue: #7fb0ff
- calibration green: #9fd29f
- warning red: #d67b70

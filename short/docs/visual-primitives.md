# Visual primitives

- `LabImage` — cinematic crop/pan of the real lab image.
- `KineticList` — fast technical detail stack.
- `Pipeline` — causal/process chain.
- `EventLanes` — typed event stream animation.
- `NodeGraph` — latent/causal graph abstraction.
- `StateTree` — counterfactual branch topology.
- `MetricGrid` — formula-heavy evaluation burst.
- `PHStripGraphic` / `CalibrationGrid` — physical measurement motifs.
